# Dark Fort wishlist

## In future we'll implement following features 

- save/load
- connect rooms one to other, and chose which door to go through
- add random encounters to explored rooms
- player will be able to wander around the dungeon
- dungeon will need to be generated on start of the game, and have minimal number of rooms
- enable selling of items in shop for half price
- shop will have limited stock for each item
- shop stock will reset for each new encounter
